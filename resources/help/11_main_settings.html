Основные настройки программы

Основные настройки приложения можно конфигурировать в дочернем окне «Предварительная настройка программы». Окно можно открыть через меню: Файл → Настройки программы.

Дочернее окно состоит из трёх основных вкладок и реализует следующие настройки приложения:

  • «Основные настройки» (Рисунок 1) - на вкладке реализуется конфигурация базовых настроек программы (цены, весовые коэффициенты и др.). Если пользователь хочет вернуть изначальные параметры, необходимо нажать на кнопку «Сбросить настройки»;
  • «Стандартные изделия» (Рисунок 2) – на вкладке реализуется конфигурация списка стандартных изделий и их цен. Для добавления нового элемента пользователь должен заполнить поля ввода и нажать на кнопку «Добавить/изменить работу». Для удаления элемента из списка пользователь должен выбрать удаляемый элемент в таблице и нажать на кнопку «Удалить элемент». Для сброса параметров, необходимо нажать на кнопку «Сброс списка работ»;
  • «Дополнительные ресурсы» (Рисунок 3) – на вкладке конфигурация списка дополнительных ресурсов и документов компании и ссылок на них. Функционал вкладке аналогичен вкладке «Стандартные изделия».

⚠ При вводе числовых параметров важно использовать точку, как знак разделить для десятичных дробей. Запятую использовать не рекомендуется в связи с форматом данных в файлах конфигурации .ini формата.

⚠ На вкладке «Основные настройки» присутствуют поля ввода с множественными значениями (оплата по счету: ООО, ИП; сложность установки; глубина гравировки; коэффициенты площади). Разделителем между значениями как раз и является запятая. Пример: «1.5, 7» – первое значение равно 1.5, второе значение равно 7.
